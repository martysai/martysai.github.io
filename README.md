# Marat Saidov Astro Prototype

Prototype redesign for Marat Saidov's personal research blog using Astro.

Highlights:

- Lab-style editorial typography: Geist + Source Serif 4 + JetBrains Mono
- Imported existing Jekyll posts into Astro content collections
- Preserved the About page, avatar, job title, and profile links
- GitHub Pages deployment workflow

Local development:

```bash
npm install
npm run dev
```

Production deployment target:

```txt
https://msaidov.com/
```

This repository deploys the Astro site to GitHub Pages for `msaidov.com`.
